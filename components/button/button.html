Buttons

Buttons are one of the most useful components in any Web Application. Glimpse UI provides different types of buttons supporting multiple sizes, colors and many more.

Solid Buttons

Create a solid button using "btn" class.

Additionally use classes such as "btn-primary", "btn-secondary", "btn-success", "btn-warn", "btn-danger", "btn-dark", for creating different buttons.

Outline Buttons

Create a outline button just by using "btn-outline" class along with specific solid buttons classes.

Link, Icon, Disabled Buttons

For creating link button add all the button classes in <a> tag.

For creating icon button use "btn-icon" class.

Use HTML "disabled" attribute to create a disabled button.

Link

Floating Icon Buttons

To create floating icon buttons use "btn-floating" class.

Different sizes of Buttons

There are three different sizes of buttons. Use "btn-sm", "btn-md", "btn-lg", classes to get small, medium and large sized buttons respectively.
