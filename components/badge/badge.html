Badges

Badges are small circles, positioned either at the top-right or bottom right of the parent components. Badge can be used to display numbers, online/offline status.

Badge on avatar

Add Badges to the avatars using "badge-container" class, along with specific badges class.

Badge on icon

Add Badges to the icons using "icon-container" class, along with "badge-container" class.
