Input

Form Inputs are really necessary element for user interaction.This is the way through which we can take login/signup data,or many more data from the user.

Standard Input

Create standard input by providing a "input-wrapper" class to the parent container of input tag.

Validation with Input

For providing validation with the input add "validation" class along with the "input-wrapper" class.

Wrong Password. Try again.

Radio Input

Create radio input using "radio" class along with the "input-wrapper" class.

To disable the input provide "disabled" class along with the "input-wrapper" class and Use HTML "disabled" attribute in the "input" tag.

Checkbox Input

Create checkbox input using "checkbox" class along with the "input-wrapper" class.

To disable the input provide "disabled" class along with the "input-wrapper" class and Use HTML "disabled" attribute in the "input" tag.
