Avatar

Avatar is mainly used to show user's profile picture usually in circular or semi-square shape.

Circle Avatar

Create circle avatar using the "circle-avatar" class.

Additionally use classes such as "avatar-lg", "avatar-md", "avatar-sm", "avatar-xs" to create large, medium, small and x-small avatars respectively.

Semi-square Avatar

Create semi square avatar using the "semi-square-avatar" class.

Text Avatar

Create text avatar using the "text-avatar" class.
