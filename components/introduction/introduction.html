Glimpse UI

Glimpse UI is an open-source pure css styling library, which helps developers quickly design beautiful and responsive web pages by using predefined class names.

How to use

To use Glimpse UI in your project, copy the link given below and paste it inside <head> tag of your html document, above all other style sheets.

Easily Customisable

Components can be easily customised. In order to override the base classes, users can add their own class along with the pre-existing classes, and style them seperately.
