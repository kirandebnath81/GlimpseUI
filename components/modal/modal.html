Modal

Modals are like pop-ups which are positioned over everything else in the document. Only way to close it, is to click the close button.

Modals can be easily created using "modal" class

You must wrap the modal inside a "modal-wrapper" class , which contains the modal and by toggling active class you can toggle the modal.

A Modal Title

Are you sure you want to delete all the videos of the playlist.This will permanatly delete your videos.
